# Jasebousbooks Book blog

## Dependencies

- [gatsbyJS](https://www.gatsbyjs.org/)
- [yarn](https://yarnpkg.com/)

## Get the repo running
1. Clone the repo
2. In the root directory run `yarn install`
3. Run `gatsby develop` for developing and watching
